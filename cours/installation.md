## **FLASK**

flask est un micro-framework python permettant de faire des applications web

---

### **Mise en place de l'environnement de developpement**

`py -3 -m venv venv`

---

### **Activation de l'environnement**

`venv\Scripts\activate`

---

### **Installation de flask**

`pip install Flask`
